# Changelog

All notable changes to the EBM30N6/FEG High Voltage Power Supply Controller project will be documented in this file.

## [Initial Release] - 2026-01-28

### Added
- Initial implementation of EBM30N6/FEG controller GUI
- Support for monitoring and controlling beam voltage, heater current, suppressor voltage, and extractor voltage
- Real-time monitoring with 2-second refresh intervals
- Safety mechanisms including heater current limiting
- Connection management with serial communication
- System status indicators for faults and warnings
- Parameter setting for all key operational values
- Support for device reset functionality
